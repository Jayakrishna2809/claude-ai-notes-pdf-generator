# Claude AI Notes PDF Generator

This project generates a clean and professional PDF containing structured study notes about Claude AI.

## Features

* Built using Python and ReportLab
* Uses Inter font for modern typography
* One section per page layout
* Includes tables, callouts, and structured notes
* Designed for easy reading and revision

## Requirements

```bash
pip install reportlab
```

## How to Run

```bash
python make_pdf_v6.py
```

## Output

Generates:

* Claude_AI_Study_Notes_Final_v11.pdf
